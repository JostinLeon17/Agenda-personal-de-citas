# ***Mantenimiento Propuesto para el Sistema de Agenda personal de citas***

## ***1. Problema: reduccion del rendimiento por carga masiva de registros en el historial***



  El sistema esta poco actualizado teniendo problemas de carga al tener el historial de citas mal optimizado.



  ***Tipo de Mantenimiento:*** Mantenimiento correctivo



  ***Acción Principal:*** Optimizar la consulta y carga del historial mediante filtros avanzados y mejoras en la consulta a la base de datos.



  ***Justificación Técnica:*** Es una mejora necesaria para evitar futuros problemas o caidas de servidores por mala optimizacion.



#### ***2. Problema: Poca perzonalicacion para la interfaz del usuario***



  El sistema no permite realizar cambios de personalizacion como los ajustes de apariencia (como el modo oscuro o claro) lo que limita la comodidad y accesibilidad del usuario.



  ***Tipo de Mantenimiento:*** Mantenimiento Perfectivo



  ***Acción Principal:*** Agregar una opcion de personalizacion que permita interfaz de modo oscuro/claro, seleccion de paleta de colores y ajustes basicos de apariencia.



  ***Justificación Técnica:*** Es requerida para la mejora de los estandares de usabilidad y accesibilidad, lo que permite que el sistema se adapte a preferencias visuales del usuario.

  El sistema deberia ser mas optimizado y util para los usuarios
